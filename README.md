# codespacee

A simple Hello World Python web application using Flask.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   python app.py
   ```

The server will start on `0.0.0.0:5000`. You can access it at `http://localhost:5000` or `http://0.0.0.0:5000`.
